    configure

    • configure(configuration): undefined

    • Configures the context for this canvas. This clears the drawing buffer to transparent black (in [$Replace the drawing buffer$]).

      Parameters

      • configuration: GPUCanvasConfiguration

        Desired configuration for the context.

      Returns undefined

    getCurrentTexture

    • getCurrentTexture(): GPUTexture

    • Get the GPUTexture that will be composited to the document by the GPUCanvasContext next. Note: The same GPUTexture object will be returned by every call to GPUCanvasContext#getCurrentTexture until "[$Expire the current texture$]" runs, even if that GPUTexture is destroyed, failed validation, or failed to allocate.

      Returns GPUTexture

    unconfigure

    • unconfigure(): undefined

    • Removes the context configuration. Destroys any textures produced while configured.

      Returns undefined
